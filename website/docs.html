Documentation & Guides

Get started with SAPP, Phasor, and the scripts in this repository. For detailed reference, visit the docs folder on GitHub.

What are SAPP & Phasor? Overview

SAPP is a powerful server extension for Halo PC/CE dedicated servers, developed by sehé. It adds Lua scripting, anti‑cheat, event hooks, player management, and more. Phasor is an earlier extension that is still compatible with many scripts.

All scripts in this repository are designed for SAPP unless noted otherwise. Phasor scripts are kept in the /phasor folder.

Installation Guide

  1. Install Halo Custom Edition (requires a valid CD key). You can find installers and patches in the README.
  2. Set up SAPP by following the official SAPP website. Extract the files into your Halo CE server directory.
  3. Download a script from this repository (e.g., gun_game.lua).
  4. Place the .lua file in your server's scripts folder.
  5. Load the script in-game using !load <scriptname> or add it to your init.txt for automatic loading.

For detailed instructions, see the Docs README.

Using Scripts – Quick Tips

  • Most scripts are activated with !load <filename> (without the .lua extension).
  • To unload: !unload <filename>
  • Check if a script is loaded: !scripts
  • Some scripts require configuration variables – edit them at the top of the .lua file before loading.
!load gun_game
!load zombies_standard
!scripts

SAPPDiscordBot Integration

The SAPPDiscordBot connects your Halo server to Discord. It sends real‑time events (joins, kills, chat) with rich embeds and has a GUI for easy setup.

Follow the repository instructions to set up the bot and link it to your Discord server.
