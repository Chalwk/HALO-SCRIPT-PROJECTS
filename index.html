Halo Script Projects

A curated collection of Lua scripts, utilities, and resources for Halo: Combat Evolved (PC/CE) dedicated servers using SAPP and Phasor. Enhance your multiplayer experience with custom games, attractive features, and powerful admin tools.

What's Inside

Custom Games

Gun Game, Zombies, Kill Confirmed, One In The Chamber, and more.

Attractive Features

Rank system, sprint, deployable mines, tactical insertion, teabagging effects.

Utility Scripts

AFK system, team shuffler, dynamic ping kicker, custom loadouts, server logger.

Integration Tools

SAPPDiscordBot – real‑time Halo server events to Discord, with rich embeds.

Open Source & Community Driven

All scripts are free to use, modify, and share. Join our Discord to contribute or get help.
